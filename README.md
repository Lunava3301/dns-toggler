# DNS Toggler 🚀

Удобный интерактивный консольный инструмент для быстрого переключения DNS-серверов в один клик. Работает как на macOS, так и на Windows.

## ✨ Особенности

*   🎨 **Красивый и современный интерфейс**: Мягкая 256-цветная палитра ANSI (пастельные тона) и плавные анимации загрузки (спиннеры).
*   ⚡ **Предустановленные DNS-шаблоны**:
    *   Google DNS (`8.8.8.8`, `8.8.4.4`)
    *   Cloudflare DNS (`1.1.1.1`, `1.0.0.1`)
    *   AdGuard DNS (блокировка рекламы: `94.140.14.14`, `94.140.15.15`)
    *   Yandex DNS (безопасный/фильтрация: `77.88.8.8`, `77.88.8.1`)
*   🔧 **Кастомный DNS**: Возможность сохранить собственный шаблон DNS и переключаться на него.
*   🔄 **Автоматический сброс**: Быстрый возврат к автоматическому получению DNS через DHCP.
*   🔍 **Автоопределение**: Скрипт сам определяет текущие DNS-настройки и отображает активный режим.
*   💻 **Мультиплатформенность**: Раздельные скрипты для macOS и Windows с автоматическим определением активного сетевого интерфейса и запросом прав администратора (UAC) на Windows.

## 🛠️ Установка и использование

### macOS (toggle_dns.py)

Скрипт управляет настройками DNS для интерфейса `Wi-Fi` (по умолчанию).

#### Способ 1: Через терминал
1. Откройте терминал и перейдите в папку со скриптом:
   ```bash
   cd /путь/к/папке/dns-toggler
   ```
2. Сделайте скрипт исполняемым (если ещё не сделано):
   ```bash
   chmod +x toggle_dns.py
   ```
3. Запустите скрипт:
   ```bash
   python3 toggle_dns.py
   ```

#### Способ 2: Запуск в один клик с рабочего стола (Toggle_DNS.command)
В репозитории есть файл `Toggle_DNS.command`. Вы можете скопировать его на Рабочий стол или в любую удобную папку. При его запуске откроется терминал и автоматически запустится скрипт.

Чтобы разрешить запуск файла `.command` в macOS, выполните один раз в терминале:
```bash
chmod +x Toggle_DNS.command
```

---

### Windows (toggle_dns_windows.py)

Скрипт автоматически определяет активный сетевой адаптер (Ethernet, Wi-Fi и др.) и запрашивает права администратора для изменения настроек сети.

#### Способ запуска:
1. Запустите скрипт с помощью Python:
   ```cmd
   python toggle_dns_windows.py
   ```
2. Согласитесь на запрос контроля учетных записей (UAC) для предоставления прав администратора.

## 📂 Структура проекта

*   `toggle_dns.py` — основной скрипт для macOS.
*   `toggle_dns_windows.py` — основной скрипт для Windows.
*   `Toggle_DNS.command` — ярлык для быстрого запуска скрипта на macOS.
*   `.gitignore` — исключение кэша и файла локальных настроек `.dns_config`.
